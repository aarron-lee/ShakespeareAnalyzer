# Play Line Counter

Counts the lines spoken by the cast in a play.

Install all related dependencies with `bundle install`, then use as follows:


In terminal:
`ruby shakespeare_analyzer.rb`

which should get back:

```
MACBETH - 719
LADY MACBETH - 265
MALCOLM - 212
MACDUFF - 180
ROSS - 135
BANQUO - 113
LENNOX - 74
DUNCAN - 70
First Witch - 62
Porter - 46
Doctor - 45
LADY MACDUFF - 41
HECATE - 39
Sergeant - 35
SIWARD - 30
...
```
